### Install

#### Clone this repo

```bash
~$ git clone https://github.com/jaeyp/electron-react-redux-calculator
```

#### Install dependencies

```bash
~$ npm install
```

---  

### Usage

#### Run the app

```bash
~$ npm run start
```

#### Build the app (automatic)

```bash
~$ npm run package
```

#### Build the app (manual)

```bash
~$ npm run build
```

#### Test the app (after `npm run build`)

```bash
~$ npm run prod
```
